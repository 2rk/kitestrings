## Installation

Add this line to your application's Gemfile:

    gem 'kitestrings'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kitestrings

Once installed run

    rails g kitestrings:install

If installed using our template file, this will already be done for you.
